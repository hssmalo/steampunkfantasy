# Races changelog

A record of deliberate balance changes to the Race data in `races/`. It captures
*why* a Unit, Model, Equipment, or Special was changed.

| Date       | Race         | Description                                                                 | Why                                                                                  |
| ---------- | ------------ | --------------------------------------------------------------------------- | ------------------------------------------------------------------------------------ |
| 14.07.2026 | Ogre         | Panther gained Terror[4]                                                    | It fit the description, and it seemed to need a slight boost.                        |
| 14.07.2026 | Ogre         | Artillery Wagon now needs a scout and to fire a shot to aim                 | Both a buff and a debuff. More fitting to how artillery works.                       |
| 14.07.2026 | Ogre         | Artillery Wagon cost from 16ip to 32ip                                      | Was overpowered.                                                                     |
| 14.07.2026 | Ogre         | Broadside Wagon cost from 16ip to 8ip                                       | Was underpowered. Still needs a main engine to move, but easier to get longer train. |
| 14.07.2026 | Ogre         | Removed Industrial Drone (but kept the regular version)                     | No need to allow that many ogre drones on the battlefield.                           |
| 08.06.2026 | Elf, Goblin  | Added special upgrade arrows (Darkelf and Ogre ranged options also touched) | Give archers more tactical variety and distinct ranged play.                         |
| 03.06.2026 | All          | Multiplied all Costs by 4                                                   | Rescaled the cost economy for finer granularity when building armies.                |
| 17.05.2026 | Abomination  | Added the Mothership Squid unit; minor cost/stat fixes across all races     | New centerpiece unit for Abomination, plus a broad balance pass.                     |
| 14.05.2026 | Ork          | Added range specials to weapons                                             | Give Ork ranged attacks their own distinct rules.                                    |
| 09.05.2026 | All          | Broad balance and fixes pass across every race                              | General tuning after testing.                                                        |
| 26.04.2026 | Abomination  | Added the Insanity Contraption to Frog Riders                               | New upgrade to flesh out the Frog Rider unit.                                        |
| 24.04.2026 | All          | Adjustments after the showcase playtest                                     | Rebalanced based on playtest feedback.                                               |
